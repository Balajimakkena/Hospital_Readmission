# 🏥 Hospital Patient Readmission Analysis

## 📌 Problem Statement
Identify key risk factors that lead to patient readmission 
within 30 days of hospital discharge using real-world data.

## 📊 Dataset
- **Source:** UCI ML Repository
- **Name:** Diabetes 130-US Hospitals (1999-2008)
- **Size:** 101,766 patients | 50 features

## 🛠️ Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
